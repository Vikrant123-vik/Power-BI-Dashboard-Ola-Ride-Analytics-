## 1. Project Title / Headline

**OLA Ride Analytics Dashboard**
An interactive Power BI dashboard built to analyze OLA ride data, focusing on ride volume, revenue, cancellations, payment modes, city-wise performance, and peak demand patterns.

---

## 2. Short Description / Purpose

The OLA Ride Analytics Dashboard provides a data-driven view of ride operations across multiple cities. It helps stakeholders monitor key performance indicators, understand customer behavior, and identify operational and revenue trends using interactive visualizations.
